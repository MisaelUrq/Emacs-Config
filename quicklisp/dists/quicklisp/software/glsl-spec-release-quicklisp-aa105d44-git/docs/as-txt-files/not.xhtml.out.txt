Name

not — logically invert a boolean vector

Declaration

bvec not(  bvec x);

 

Parameters

x

    Specifies the vector to be inverted.

Description

not logically inverts the boolean vector x. It returns a new boolean vector for
which each element i is computed as !x[i].
