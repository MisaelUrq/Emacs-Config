Name

length — calculate the length of a vector

Declaration

float length(  genType x);

 

double length(  genDType x);

 

Parameters

x

    Specifies a vector of which to calculate the length.

Description

length returns the length of the vector. i.e., x[0] 2 + x[1] 2 + …
